# 秦根霖命理教学 - 网页版

## 简介

这是秦根霖命理教学的网页版本，与小程序版使用同一套数据和后台管理系统。

## 快速开始

### 方式一：直接打开（本地使用）

1. 用浏览器直接打开 `index.html` 文件即可使用
2. 支持 Chrome、Safari、Firefox 等现代浏览器
3. 手机和电脑都可以使用

### 方式二：本地服务器预览

如果你安装了 Node.js，可以运行：

```bash
cd web
npx serve
```

然后在浏览器中打开显示的地址（通常是 http://localhost:3000）

### 方式三：部署到线上

可以部署到以下免费平台：

- **GitHub Pages** - 免费，适合公开访问
- **Vercel** - 免费，部署简单
- **Netlify** - 免费，支持拖拽上传

## 数据同步

网页版和小程序版共享同一套数据：

- 数据文件：`web/data.js`（与小程序的 `utils/data.js` 保持一致）
- 后台管理系统：`admin/` 目录
- 在后台修改数据后，记得把 `utils/data.js` 复制到 `web/data.js`

## 功能特性

- ✅ 课程列表展示
- ✅ 课程详情阅读
- ✅ 题目练习（选择题）
- ✅ 即时反馈和解析
- ✅ 成绩统计
- ✅ 响应式设计（手机/电脑自适应）
- ✅ 美观的渐变色界面

## 与小程序版的区别

| 特性 | 网页版 | 小程序版 |
|------|--------|----------|
| 审核 | 无需审核 | 需要微信审核 |
| 访问方式 | 直接链接 | 小程序码/搜索 |
| 分享 | 发链接即可 | 分享小程序 |
| 数据 | 同一套 | 同一套 |
| 后台 | 同一套 | 同一套 |

## 建议

- **优先使用网页版** - 无需审核，随时可用
- **小程序版备选** - 如果审核通过，可以作为补充
- **双版本并行** - 两个版本都提供给学员，让他们选择
